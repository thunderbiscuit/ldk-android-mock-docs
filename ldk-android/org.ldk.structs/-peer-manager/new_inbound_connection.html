new_inbound_connection

open fun new_inbound_connection(descriptor: SocketDescriptor): Result_NonePeerHandleErrorZ
Content copied to clipboard

Indicates a new inbound connection has been established. May refuse the connection by returning an Err, but will never write bytes to the remote end (outbound connector always speaks first). Note that if an Err is returned here you MUST NOT call socket_disconnected for the new descriptor but must disconnect the connection immediately. Panics if descriptor is duplicative with some other descriptor which has not yet been [`socket_disconnected()`]. [`socket_disconnected()`]: PeerManager::socket_disconnected
