Content copied to clipboard

Indicates that there is room to write data to the given socket descriptor. May return an Err to indicate that the connection should be closed. May call [`send_data`] on the descriptor passed in (or an equal descriptor) before returning. Thus, be very careful with reentrancy issues! The invariants around calling [`write_buffer_space_avail`] in case a write did not fully complete must still hold - be ready to call `[write_buffer_space_avail`] again if a write call generated here isn't sufficient!
